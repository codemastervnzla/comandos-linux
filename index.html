Comando "ls"

Se utiliza para mostrar el contenido de un directorio (carpeta), como por ejemplo mas directorios dentro de este, archivos o ficheros, fotos, etc.

Comando "cd"

Nos permite movernos entre los distintos directorios del sitema operativo.

Comando "du"

Este comando nos muestra el espacio que ocupa en el disco un fichero o un directorio

$ du -h directorio

Comando "cp"

Este comando nos permite copiar un fichero o un directorio.

$ cp fichero.txt directotio/
